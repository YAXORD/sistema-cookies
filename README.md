EN# 🍪 Cookie Management System (Consent Mode V2)

A lightweight, professional cookie manager designed to comply with Google and InfoTrust regulations.

### 🚀 Key Features
* **G100/G111 Support:** Native implementation of Google Consent Mode V2.
* **Injectable:** Ready to be tested on any website via the browser console.
* **Fully Customizable:** Includes an integrated banner and a detailed settings menu.

### 🛠️ How to Test the Demo
1. Copy the injector code.
2. Open the browser console (**F12**) on any website.
3. Paste the code and press **Enter**.

> [!NOTE]
> This demo works best on websites without existing complex cookie apps or systems. I am currently working on improving compatibility and adding new features!


ES# 🍪 Sistema de Gestión de Cookies (Consent Mode V2)

Un gestor de cookies ligero y profesional diseñado para cumplir con las normativas de Google e InfoTrust.

### 🚀 Características Principales
* **Soporte G100/G111:** Implementación nativa de Google Consent Mode V2.
* **Inyectable:** Listo para ser probado en cualquier sitio web a través de la consola del navegador.
* **Totalmente Personalizable:** Incluye un banner integrado y un menú de ajustes detallado.

### 🛠️ Cómo Probar la Demo
1. Copia el código del inyector.
2. Abre la consola del navegador (**F12**) en cualquier sitio web.
3. Pega el código y pulsa **Enter**.

> [!NOTE]
> Esta demo funciona mejor en sitios web que no tengan sistemas de cookies complejos instalados. ¡Actualmente estoy trabajando en mejorar la compatibilidad y añadir nuevas funciones!
